17. 音の脱落「リダクション（短縮）」②

wanna, gonna, hafta などの口語表現

なぜリダクション（短縮）が重要なのか？

英語では、日常会話で単語が短縮されることが頻繁にあります。これを「リダクション（短縮）」と言います。これを理解することで、ネイティブの自然な会話を聞き取れるようになります。

よくある間違い

want to（〜したい）

カタカナ読み：「ウォント トゥ」（すべての音を発音）

実際の音：[ˈwɑntə] → 「ウォナ」（「want to」が「wanna」に短縮）

going to（〜するつもり）

カタカナ読み：「ゴーイング トゥ」（すべての音を発音）

実際の音：[ˈgɔnə] → 「ゴナ」（「going to」が「gonna」に短縮）

have to（〜しなければならない）

カタカナ読み：「ハブ トゥ」（すべての音を発音）

実際の音：[ˈhæftə] → 「ハフタ」（「have to」が「hafta」に短縮）

リダクション（短縮）をマスターすることで、ネイティブの自然な会話を聞き取れる・話せるようになります。

リダクション（短縮）とは？

リダクション（短縮）は、日常会話で単語が短縮される現象です。主に、機能語が短縮されることが多いです。

リダクション（短縮）の特徴

1. 日常会話で頻繁に使われる

リダクション（短縮）は、日常会話で頻繁に使われます。フォーマルな場面では使わないこともあります。

例：I want to go → 「I wanna go」

2. 機能語が短縮される

機能語（want to, going to, have to など）が短縮されることが多いです。

例：want to → 「wanna」

3. 書き言葉では使わない

リダクション（短縮）は、話し言葉で使われます。書き言葉では使わないことが多いです。

リダクション（短縮）の例

リダクション（短縮）の具体例を見てみましょう。

1. want to → wanna

書き方：I want to go

カタカナ読み：「アイ ウォント トゥ ゴー」

実際の音：[aɪ ˈwɑntə goʊ] → 「アイ ウォナ ゴー」（「want to」が「wanna」に短縮）

「want to」→「wanna」

2. going to → gonna

書き方：I'm going to go

カタカナ読み：「アイム ゴーイング トゥ ゴー」

実際の音：[aɪm ˈgɔnə goʊ] → 「アイム ゴナ ゴー」（「going to」が「gonna」に短縮）

「going to」→「gonna」

3. have to → hafta

書き方：I have to go

カタカナ読み：「アイ ハブ トゥ ゴー」

実際の音：[aɪ ˈhæftə goʊ] → 「アイ ハフタ ゴー」（「have to」が「hafta」に短縮）

「have to」→「hafta」

4. got to → gotta

書き方：I got to go

カタカナ読み：「アイ ゴット トゥ ゴー」

実際の音：[aɪ ˈgɑtə goʊ] → 「アイ ガタ ゴー」（「got to」が「gotta」に短縮）

「got to」→「gotta」

リダクション（短縮）のルール

リダクション（短縮）の基本的なルールを学びましょう。

基本的なルール

1. 日常会話で使われる

リダクション（短縮）は、日常会話で使われます。フォーマルな場面では使わないこともあります。

例：I want to go → 「I wanna go」（カジュアル）

2. 機能語が短縮される

機能語（want to, going to, have to など）が短縮されることが多いです。

例：want to → 「wanna」

3. 書き言葉では使わない

リダクション（短縮）は、話し言葉で使われます。書き言葉では使わないことが多いです。
