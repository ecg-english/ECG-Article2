20. イントネーションの基本②（上げ調子）

Yes/No疑問文、聞き返し。文末を上げる

なぜ上げ調子が重要なのか？

英語では、Yes/No疑問文や聞き返しでは、文末で音程を上げるイントネーションを使います。これを「上げ調子（Rising Intonation）」と言います。

上げ調子が重要な理由

1. 英語らしい自然なイントネーション

上げ調子をマスターすることで、英語らしい自然なイントネーションができます。これが「英語らしく聞こえる」最大の秘密です。

2. 疑問の表現

上げ調子は、疑問を表現する上で重要な役割を果たします。Yes/No疑問文や聞き返しでは、上げ調子を使うことで、疑問や確認を表します。

3. 聞き取りの鍵

上げ調子を理解することで、ネイティブの話す英語を聞き取りやすくなります。自分で正しく発音できるイントネーションは聞き取れるようになります。

上げ調子とは？

上げ調子（Rising Intonation）は、文末で音程を上げるイントネーションです。Yes/No疑問文や聞き返しで使われます。正式には「Rising Intonation」と呼ばれます。

上げ調子の特徴

1. 文末で音程を上げる

文末で音程を上げます。これが「上げ調子」です。音程が低い状態から高い状態へと上がります。

例：Do you like apples? → 文末を上げる

「apples」の部分で音程を上げます。低い音から高い音へと上がります。

2. Yes/No疑問文で使われる

Yes/No疑問文では、上げ調子を使います。疑問や確認を表します。

例：Do you like apples? → 上げ調子

Yes/No疑問文では、上げ調子を使うことで、疑問や確認を表します。

3. 聞き返しで使われる

聞き返しでは、上げ調子を使います。確認や驚きを表します。

例：Really? → 上げ調子

聞き返しでは、上げ調子を使うことで、確認や驚きを表します。

4. 丁寧な表現で使われる

丁寧な表現では、上げ調子を使うことがあります。親しみやすさや丁寧さを表します。

例：Would you like some coffee? → 上げ調子

丁寧な表現では、上げ調子を使うことで、親しみやすさや丁寧さを表します。

上げ調子の音声学的特徴

音声学的特徴

  • 文末の音節で音程を上げます

上げ調子と下げ調子の違い

上げ調子と下げ調子の違いを明確に理解しましょう。文の種類によって、使い分けが重要です。

上げ調子（Rising Intonation）

  • 文末で音程を上げる
  • Yes/No疑問文で使われる
  • 聞き返しで使われる
  • 疑問や確認を表す
  • 例：Do you like apples?

下げ調子（Falling Intonation）

  • 文末で音程を下げる
  • 肯定文で使われる
  • Wh疑問文で使われる
  • 確信や完了を表す
  • 例：I like apples.
