20. イントネーションの基本②（上げ調子）

Yes/No疑問文、聞き返し。文末を上げる

なぜ上げ調子が重要なのか？

英語では、Yes/No疑問文や聞き返しでは、文末で音程を上げるイントネーションを使います。これを「上げ調子」と言います。

よくある間違い

Yes/No疑問文：Do you like apples?（あなたはリンゴが好きですか？）

カタカナ読み：「ドゥ ユー ライク アップルズ」（すべて同じ高さで読む）

実際の音：文末を上げる（上げ調子）

聞き返し：Really?（本当？）

カタカナ読み：「リアリー」（すべて同じ高さで読む）

実際の音：文末を上げる（上げ調子）

上げ調子をマスターすることで、英語らしい自然なイントネーションと疑問の表現ができるようになります。

上げ調子とは？

上げ調子は、文末で音程を上げるイントネーションです。Yes/No疑問文や聞き返しで使われます。

上げ調子の特徴

1. 文末で音程を上げる

文末で音程を上げます。これが「上げ調子」です。

例：Do you like apples? → 文末を上げる

2. Yes/No疑問文で使われる

Yes/No疑問文では、上げ調子を使います。

例：Do you like apples? → 上げ調子

3. 聞き返しで使われる

聞き返しでは、上げ調子を使います。

例：Really? → 上げ調子

上げ調子の例

上げ調子の具体例を見てみましょう。

例1：Yes/No疑問文

書き方：Do you like apples?

イントネーション：文末を上げる（上げ調子）

「Do you like apples?」→ 文末の「apples」で音程を上げる

例2：聞き返し

書き方：Really?

イントネーション：文末を上げる（上げ調子）

「Really?」→ 文末で音程を上げる

例3：Yes/No疑問文（Are you）

書き方：Are you okay?

イントネーション：文末を上げる（上げ調子）

「Are you okay?」→ 文末の「okay」で音程を上げる

例4：Yes/No疑問文（Can you）

書き方：Can you help me?

イントネーション：文末を上げる（上げ調子）

「Can you help me?」→ 文末の「me」で音程を上げる

上げ調子と下げ調子の違い

上げ調子と下げ調子の違いを明確に理解しましょう。

上げ調子

  • 文末で音程を上げる
  • Yes/No疑問文で使われる
  • 聞き返しで使われる
  • 例：Do you like apples?

下げ調子

  • 文末で音程を下げる
  • 肯定文で使われる
  • Wh疑問文で使われる
  • 例：I like apples.

実践練習：上げ調子をマスターしよう

練習1：上げ調子を意識して発音

ステップ1：文の種類を確認

Yes/No疑問文、聞き返しなど、文の種類を確認します。

ステップ2：文末で音程を上げる

文末で音程を上げます。これが「上げ調子」です。

ステップ3：自然に発音

上げ調子を意識しながら、自然に発音します。

重要：文末で音程を上げる（上げ調子）

重要ポイントまとめ

1. 上げ調子の特徴

文末で音程を上げるイントネーションです。Yes/No疑問文や聞き返しで使われます。

2. Yes/No疑問文で使われる

Yes/No疑問文では、上げ調子を使います。例：Do you like apples? → 文末を上げる

3. 聞き返しで使われる

聞き返しでは、上げ調子を使います。例：Really? → 文末を上げる

4. 上げ調子と下げ調子の違い

上げ調子は文末で音程を上げる、下げ調子は文末で音程を下げる。文の種類によって使い分けます。

5. 練習が大切

上げ調子をマスターするには、繰り返し練習することが重要です。フレーズで練習しましょう。
