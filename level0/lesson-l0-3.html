3. 日本人の宿敵②「TH（無声音・有声音）」

舌を噛む勇気。Thank you / This

なぜTHが難しいのか？

THは、日本人にとって最も難しい音の一つです。その理由は、日本語にはTHの音が存在しないからです。多くの日本人は、THを「サ行」や「ザ行」で代用してしまいます。

よくある間違い

thank you（ありがとう）

カタカナ読み：「サンキュー」→ THが「サ」になってしまう

正しくは：[θæŋk ju]（舌を噛む「サ」）

this（これ）

カタカナ読み：「ディス」→ THが「ディ」になってしまう

正しくは：[ðɪs]（舌を噛む「ディ」）

think（考える）とsink（沈む）

カタカナ読み：どちらも「シンク」→ 意味が全く違うのに同じ音に聞こえる！

THとSを区別できないと、会話で大きな誤解を生む可能性があります。

they（彼ら）とday（日）

カタカナ読み：「ゼイ」と「デイ」→ THが「ゼ」や「デ」になってしまう

THとDを区別できないと、意味が変わってしまいます。

しかし、THは「舌を噛む」という正しい方法で練習すれば、必ずマスターできます。このレッスンで、その方法を学びましょう。

日本語にはTHの音がない

日本語にはTHの音が存在しないため、多くの日本人はTHを「サ行」や「ザ行」で代用してしまいます。しかし、THとS/Zは全く違う音です。

🇯🇵 日本語の「サ行」

  • 舌先を上の歯茎に当てる
  • THの音は存在しない
  • THを「サ」で代用

🇺🇸 英語のTH [θ]

  • 舌先を歯の間に挟む
  • 息だけの音
  • 「サ」とは全く違う

🇺🇸 英語のTH [ð]

  • 舌先を歯の間に挟む
  • 声を出す音
  • 「ザ」とは全く違う

THには2種類ある

THには、無声音のTH [θ]と有声音のTH [ð]の2種類があります。どちらも「舌を噛む」音ですが、声帯を震わせるかどうかが違います。

無声音のTH [θ]

声帯を震わせない（息だけの音）

特徴：

  • 息だけの音（声帯を震わせない）
  • 「サ」に近いが、舌を噛む
  • 舌先を歯の間に挟む
  • 例：think, thank, three, thing, through

使用頻度：名詞、動詞、形容詞など、内容語に多く使われます。

有声音のTH [ð]

声帯を震わせる（声を出す）

特徴：

  • 声を出す音（声帯を震わせる）
  • 「ザ」に近いが、舌を噛む
  • 舌先を歯の間に挟む
  • 例：this, that, the, they, there

使用頻度：代名詞、冠詞、前置詞など、機能語に多く使われます。

どちらを使うかの見分け方

無声音のTH [θ]が使われる場合

  • 名詞：thing, thought, thumb
  • 動詞：think, thank, throw
  • 形容詞：thick, thin, thirsty
  • 数詞：three, thirteen, thirty

有声音のTH [ð]が使われる場合

  • 代名詞：this, that, they, them
  • 冠詞：the
  • 前置詞：with, without
  • 接続詞：though, although

無声音のTH [θ]：息だけの音（詳細版）

無声音のTH [θ]は、「舌を噛んで息だけを出す」音です。声帯を震わせません。多くの日本人が「サ」で代用してしまいますが、THとSは全く違う音です。

無声音のTH [θ]の発音のコツ（詳細）

ステップ1：舌の位置

舌先を上の前歯と下の前歯の間に軽く挟む。舌先を少しだけ出してもOK。

重要：舌を「噛む」というより「挟む」感じ。痛くない程度に。

舌先を出しすぎると不自然になりますが、少し出る程度なら問題ありません。鏡を見ながら確認しましょう。

ステップ2：息を出す

舌を挟んだまま、息だけを出す。声帯を震わせない（「サ」の無声音に近い）。

練習：鏡を見ながら「スー」と言うが、舌を歯の間に挟む。息だけが出ることを確認。

手を口の前に当てて、息だけが出ていることを確認しましょう。声帯が震えていないことも確認できます。

ステップ3：Sとの違い

Sは舌先を上の歯茎に当てるが、THは舌先を歯の間に挟む。これが最大の違い。

SとTHを交互に発音して、舌の位置の違いを感じましょう。

無声音のTH [θ]の発音チェックポイント

✓ 舌先が歯の間に挟まれているか？

舌先を上の前歯と下の前歯の間に軽く挟んでいる。

✓ 息だけが出ているか？

声帯を震わせず、息だけが出ている。

✓ Sと違う音になっているか？

S（舌先を歯茎に当てる）とは違う音になっている。

✓ 鏡で確認できるか？

鏡を見ながら、舌先が歯の間に挟まれているのが見える。

無声音のTH [θ]でよくある間違い

間違い1：舌を出さない

舌を歯の間に挟まず、舌先を歯茎に当ててしまうと、Sになってしまいます。

修正方法：舌先を歯の間に挟むことを意識する。鏡を見ながら確認する。

間違い2：舌を出しすぎる

舌を出しすぎると、不自然な音になってしまいます。

修正方法：舌先を少しだけ出す程度にする。軽く挟む感じ。

間違い3：声を出してしまう

無声音のTHなのに、声を出してしまうと、有声音のTHになってしまいます。

修正方法：息だけを出すことを意識する。手を口の前に当てて、息だけが出ていることを確認する。

有声音のTH [ð]：声を出す音（詳細版）

有声音のTH [ð]は、「舌を噛んで声を出す」音です。声帯を震わせます。多くの日本人が「ザ行」や「ダ行」で代用してしまいますが、THとZ/Dは全く違う音です。

有声音のTH [ð]の発音のコツ（詳細）

ステップ1：舌の位置

無声音のTHと同じ。舌先を上の前歯と下の前歯の間に軽く挟む。

無声音のTHと同じ舌の位置で、声を出すだけです。

ステップ2：声を出す

舌を挟んだまま、声を出す。声帯を震わせる（「ザ」の有声音に近い）。

練習：鏡を見ながら「ズー」と言うが、舌を歯の間に挟む。声が出ることを確認。

手を喉に当てて、声帯が震えていることを確認しましょう。

ステップ3：Z/Dとの違い

Zは舌先を上の歯茎に当てるが、THは舌先を歯の間に挟む。Dは舌先を上の歯茎に当てて破裂させるが、THは舌先を歯の間に挟んで摩擦音を出す。これが最大の違い。

Z/DとTHを交互に発音して、舌の位置の違いを感じましょう。

有声音のTH [ð]の発音チェックポイント

✓ 舌先が歯の間に挟まれているか？

舌先を上の前歯と下の前歯の間に軽く挟んでいる。

✓ 声が出ているか？

声帯を震わせて、声を出している。

✓ Z/Dと違う音になっているか？

Z（舌先を歯茎に当てる）やD（舌先を歯茎に当てて破裂）とは違う音になっている。

✓ 鏡で確認できるか？

鏡を見ながら、舌先が歯の間に挟まれているのが見える。

有声音のTH [ð]でよくある間違い

間違い1：舌を出さない

舌を歯の間に挟まず、舌先を歯茎に当ててしまうと、Zになってしまいます。

修正方法：舌先を歯の間に挟むことを意識する。鏡を見ながら確認する。

間違い2：Dで代用する

THをDで代用してしまうと、意味が変わってしまいます（例：they → day）。

修正方法：舌先を歯の間に挟んで、摩擦音を出す。Dは破裂音なので、音の質が違います。

間違い3：声を出さない

有声音のTHなのに、声を出さないと、無声音のTHになってしまいます。

修正方法：声を出すことを意識する。手を喉に当てて、声帯が震えていることを確認する。

THとS/Zの違いを明確に（詳細版）

THとS/Zの違いを明確にするために、対比して練習しましょう。違いを理解することで、正しい発音ができるようになります。

THとS/Zの比較（詳細）

TH [θ] / [ð]：舌を歯の間に挟む

  • 舌先を上の前歯と下の前歯の間に挟む
  • 舌先が少し見える
  • 摩擦音（息や声が舌と歯の間を通る）
  • 例：think, this

S / Z：舌先を歯茎に当てる

  • 舌先を上の歯茎に当てる
  • 舌先は見えない
  • 摩擦音（息や声が舌と歯茎の間を通る）
  • 例：sink, zip

違いを感じる練習

練習1：舌の位置を確認

鏡を見ながら、THとSを交互に発音して、舌の位置の違いを確認しましょう。

  • TH：舌先が歯の間に挟まれている
  • S：舌先が歯茎に当たっている

練習2：音の違いを感じる

THとSを交互に発音して、音の違いを感じましょう。

  • TH：舌と歯の間を通る音
  • S：舌と歯茎の間を通る音

THの位置による違い

THは、単語の中での位置によって、発音が少し変わることがあります。これを理解することで、より自然な発音ができます。

THの位置による違い

語頭のTH

語頭のTHは、はっきりと発音します。舌をしっかりと歯の間に挟みます。

例：think [θɪŋk]、this [ðɪs]

語中のTH

語中のTHも、舌を歯の間に挟んで発音します。ただし、語頭ほど強くはありません。

例：something [ˈsʌmθɪŋ]、mother [ˈmʌðər]

語尾のTH

語尾のTHも、舌を歯の間に挟んで発音します。特に注意が必要です。

例：bath [bæθ]、with [wɪð]

実践練習：THをマスターしよう

練習1：無声音のTH [θ]（基礎編）

ステップ1：舌の準備

舌先を上の前歯と下の前歯の間に軽く挟む。

鏡を見ながら、舌先が歯の間に挟まれていることを確認しましょう。

ステップ2：息を出す

舌を挟んだまま、息だけを出す。声帯を震わせない。

手を口の前に当てて、息だけが出ていることを確認しましょう。

ステップ3：Sとの違いを確認

SとTHを交互に発音して、舌の位置の違いを確認する。

単語で練習（語頭）

think, thank, three, thing, through, thickを繰り返し発音。

各単語で、THの音を意識して発音しましょう。

単語で練習（語中・語尾）

something, bath, month, healthを繰り返し発音。

語中・語尾のTHも、舌を歯の間に挟んで発音します。

練習2：有声音のTH [ð]（基礎編）

ステップ1：舌の準備

舌先を上の前歯と下の前歯の間に軽く挟む。

無声音のTHと同じ舌の位置です。

ステップ2：声を出す

舌を挟んだまま、声を出す。声帯を震わせる。

手を喉に当てて、声帯が震えていることを確認しましょう。

ステップ3：Z/Dとの違いを確認

Z/DとTHを交互に発音して、舌の位置の違いを確認する。

単語で練習（語頭）

this, that, they, there, then, thoughを繰り返し発音。

各単語で、THの音を意識して発音しましょう。

単語で練習（語中・語尾）

the, with, mother, fatherを繰り返し発音。

語中・語尾のTHも、舌を歯の間に挟んで発音します。

練習3：THとSの対比

ペアで練習（基本）

  • think → sink
  • thank → sank
  • three → see
  • thick → sick

交互に発音して、違いを明確にしましょう。

練習のコツ

  • 鏡を見ながら、舌の位置を確認する
  • TH：舌先が歯の間に挟まれている
  • S：舌先が歯茎に当たっている
  • ゆっくりと、はっきりと発音する

練習4：THとZ/Dの対比

ペアで練習

  • this → zoo（THとZ）
  • they → day（THとD）
  • then → den（THとD）

交互に発音して、違いを明確にしましょう。

練習5：THを含む文で練習

短い文で練習

  • Thank you very much.（どうもありがとう）
  • I think this is good.（これは良いと思います）
  • They are there.（彼らはそこにいます）

文の中でTHを正しく発音する練習をしましょう。

長い文で練習

  • I think that they are there with their mother.（私は、彼らが母親と一緒にそこにいると考えます）
  • Thank you for thinking about this thing.（このことについて考えてくれてありがとう）

長い文でも、THを正しく発音する練習をしましょう。

よくある間違いとその修正方法

THの発音でよくある間違いと、その修正方法を学びましょう。

間違い1：THをSで代用する

症状

THをSで代用してしまう。thinkがsinkに聞こえる。

修正方法

  1. 鏡を見ながら、舌先を歯の間に挟む
  2. 舌先が歯の間に挟まれていることを確認する
  3. S（舌先を歯茎に当てる）とTH（舌先を歯の間に挟む）を交互に発音する
  4. 違いを感じるまで繰り返す

間違い2：THをZ/Dで代用する

症状

THをZ/Dで代用してしまう。theyがdayに聞こえる。

修正方法

  1. 鏡を見ながら、舌先を歯の間に挟む
  2. Z/D（舌先を歯茎に当てる）とTH（舌先を歯の間に挟む）を交互に発音する
  3. THは摩擦音、Dは破裂音であることを意識する
  4. 違いを感じるまで繰り返す

間違い3：舌を出さない

症状

舌を歯の間に挟まず、舌先を歯茎に当ててしまう。SやZになってしまう。

修正方法

  1. 鏡を見ながら、舌先を歯の間に挟むことを意識する
  2. 舌先が少し見える程度にする
  3. 痛くない程度に軽く挟む
  4. その状態で発音する

間違い4：無声音と有声音を混同する

症状

無声音のTHで声を出してしまう、または有声音のTHで声を出さない。

修正方法

  1. 無声音のTH：息だけを出す（手を口の前に当てて確認）
  2. 有声音のTH：声を出す（手を喉に当てて確認）
  3. 交互に発音して、違いを感じる

セルフチェック方法

自分のTHの発音をチェックする方法を学びましょう。

チェック方法

1. 鏡を見ながら発音

鏡を見ながら発音することで、舌の位置を確認できます。

  • TH：舌先が歯の間に挟まれているか？
  • S/Z：舌先が歯茎に当たっているか？
  • 違いが明確に見えるか？

2. 録音して聞き返す

自分の声を録音して聞き返すことで、客観的に発音をチェックできます。

  • ネイティブの音声と比較する
  • THとS/Zの音の違いを感じる
  • 改善点を見つける

3. 手を当てて確認

手を当てることで、体の動きを確認できます。

  • 無声音のTH：手を口の前に当てて、息だけが出ているか確認
  • 有声音のTH：手を喉に当てて、声帯が震えているか確認

4. 対比練習で確認

THとS/Zのペアを交互に発音して、違いを確認します。

  • think/sink、this/zooなどを交互に発音
  • 違いが明確に感じられるか確認

聞き取りの練習

THを正しく発音するだけでなく、聞き取ることも重要です。聞き取りの練習もしましょう。

聞き取りの練習方法

1. ネイティブの音声を聞く

ネイティブの音声を聞いて、THとS/Zの違いを感じましょう。

  • think/sink、this/zooなどのペアを聞く
  • THとS/Zの音の違いを感じる
  • 何度も繰り返し聞く

2. 聞き取りクイズ

THとS/Zを含む単語を聞いて、どちらかを判断する練習をしましょう。

  • thinkかsinkか？
  • thisかzooか？
  • theyかdayか？

3. シャドーイング

継続的な練習のコツ

THをマスターするには継続が最も重要です。続けやすいコツを学びましょう。

継続のコツ

1. 毎日少しずつ練習する

1日5分でも10分でも、毎日続けることが大切です。完璧を目指さず、継続を優先しましょう。

2. 鏡を見ながら練習する

鏡を見ながら練習することで、舌の位置を確認できます。

3. 対比練習を繰り返す

THとS/Zのペアを交互に発音することで、違いを明確にできます。

4. 楽しみながら練習する

好きな歌を歌ったり、好きな映画のセリフを真似したり、楽しみながら練習しましょう。

5. 小さな目標を設定する

「1週間続ける」「1つのペアをマスターする」など、小さな目標を設定して達成感を味わいましょう。

重要ポイントまとめ

1. THには2種類ある

無声音のTH [θ]（息だけ）と有声音のTH [ð]（声を出す）。どちらも舌を歯の間に挟む。

2. 舌を噛む（挟む）

舌先を上の前歯と下の前歯の間に軽く挟む。痛くない程度に。これがTHの最大の特徴。

3. S/Z/Dとの違い

S/Zは舌先を歯茎に当てるが、THは舌先を歯の間に挟む。Dは破裂音だが、THは摩擦音。これが最大の違い。

4. 無声音と有声音の見分け方

無声音のTHは名詞、動詞、形容詞などに多く、有声音のTHは代名詞、冠詞、前置詞などに多く使われます。

5. 鏡を見ながら練習

THの発音を練習するときは、必ず鏡を見ながら、舌が歯の間に挟まれていることを確認しましょう。

6. 対比練習が効果的

THとS/Zのペア（think/sink、this/zooなど）を交互に発音することで、違いを明確にできます。

7. 継続が大切

THは難しい音ですが、毎日練習すれば必ずマスターできます。あきらめずに続けましょう。
