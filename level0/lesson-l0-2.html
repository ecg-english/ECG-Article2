2. 日本人の宿敵①「RとL」

舌を巻くR、歯茎に当てるL。Right / Lightの撃退

なぜRとLが難しいのか？

RとLは、日本人にとって最も難しい音の一つです。その理由は、日本語にはRとLの区別がないからです。日本語の「ラ行」は、実はRでもLでもない、中間的な音なのです。

よくある間違い

right（右）とlight（光）

カタカナ読み：どちらも「ライト」→ ネイティブには区別がつかない！

正しく発音すると、全く違う音になります。

rock（岩）とlock（鍵）

カタカナ読み：どちらも「ロック」→ 意味が全く違うのに同じ音に聞こえる！

RとLを区別できないと、会話で大きな誤解を生む可能性があります。

fry（揚げる）とfly（飛ぶ）

カタカナ読み：どちらも「フライ」→ 料理と飛行が同じ音に聞こえる！

文脈で判断できる場合もありますが、発音が正しければ誤解は避けられます。

しかし、RとLは正しい方法で練習すれば、必ずマスターできます。このレッスンで、その方法を学びましょう。

日本語の「ラ行」と英語のR/Lの違い

日本語の「ラ行」は、実はRでもLでもない、独特の音です。これを理解することが、RとLをマスターする第一歩です。

🇯🇵 日本語の「ラ行」

  • 舌先が上あごに軽く触れる
  • RとLの中間的な音
  • 1種類の音しかない

🇺🇸 英語のR

  • 舌先はどこにも当てない
  • 舌根を上げる
  • 口をすぼめる

🇺🇸 英語のL

  • 舌先を歯茎にしっかり当てる
  • 舌根は下げる
  • 口は自然な形

Rの発音：舌を巻く（詳細版）

Rは「舌を巻く」音です。しかし、実際には舌を「巻く」のではなく、舌の後ろの部分（舌根）を上げることが重要です。この誤解が、Rを難しくしている原因の一つです。

Rの発音のコツ（詳細）

ステップ1：舌の位置

舌先を下の歯の裏に軽く当てる。舌の後ろの部分（舌根）を上に上げる。

重要：舌先はどこにも当てない！

多くの人が舌先を上あごに当ててしまいますが、これでは日本語の「ラ」になってしまいます。

ステップ2：口の形

口をすぼめる（「ウ」の形）。唇を前に突き出す。

口をすぼめることで、Rの音がより明確になります。鏡を見ながら、唇が前に突き出ていることを確認しましょう。

ステップ3：声を出す

「ルー」という感じで声を出す。ただし、舌先はどこにも当てない。

練習：鏡を見ながら「ルー」と言って、舌先が動かないことを確認。

舌先が動いてしまう場合は、舌根を上げることに集中しましょう。

Rの発音チェックポイント

✓ 舌先が動いていないか？

舌先は下の歯の裏に軽く当てたまま、動かさない。

✓ 舌根が上がっているか？

舌の後ろの部分が上に上がっている感覚がある。

✓ 口がすぼまっているか？

唇が前に突き出ている。

✓ 音が響いているか？

「ルー」という感じで、深く響く音が出ている。

Rの発音でよくある間違い

間違い1：舌先を上あごに当てる

舌先を上あごに当ててしまうと、日本語の「ラ」になってしまいます。

修正方法：舌先を下の歯の裏に当て、舌根を上げることに集中する。

間違い2：舌を無理に巻こうとする

舌を無理に巻こうとすると、不自然な音になってしまいます。

修正方法：舌を巻くのではなく、舌根を上げることに集中する。

間違い3：口をすぼめない

口をすぼめないと、Rの音が不明瞭になります。

修正方法：口をすぼめて、唇を前に突き出す。

Lの発音：歯茎に当てる（詳細版）

Lは「舌先を上の歯茎に当てる」音です。日本語の「ラ行」に近いですが、舌先をしっかりと上の歯茎に当てることが重要です。また、Lには「明るいL」と「暗いL」の2種類があります。

Lの発音のコツ（詳細）

ステップ1：舌の位置

舌先を上の歯茎（前歯の後ろ）にしっかりと当てる。

重要：舌先を離さない！

舌先を軽く当てるのではなく、しっかりと当てることが重要です。これが日本語の「ラ」との違いです。

ステップ2：口の形

口は自然な形。特にすぼめない。

Rとは違い、Lは口をすぼめません。自然な形で発音します。

ステップ3：声を出す

舌先を歯茎に当てたまま「ル」という感じで声を出す。

練習：鏡を見ながら「ル」と言って、舌先が歯茎に当たっていることを確認。

舌先が離れてしまう場合は、舌先を歯茎に当てることに集中しましょう。

明るいLと暗いL

Lには「明るいL」と「暗いL」の2種類があります。位置によって発音が少し変わります。

明るいL（語頭・語中）

語頭や語中にあるL。舌先を歯茎に当てて発音。

例：light [laɪt]、hello [həˈloʊ]

暗いL（語尾・子音の前）

語尾や子音の前にあるL。舌先を歯茎に当て、舌根を少し上げる。

例：ball [bɔl]、help [help]

Lの発音チェックポイント

✓ 舌先が歯茎に当たっているか？

舌先を上の歯茎にしっかりと当てている。

✓ 舌先が離れていないか？

発音中、舌先が歯茎から離れない。

✓ 口が自然な形か？

口をすぼめず、自然な形で発音している。

✓ 音がはっきりしているか？

「ル」という感じで、はっきりとした音が出ている。

Lの発音でよくある間違い

間違い1：舌先を軽く当てる

舌先を軽く当ててしまうと、日本語の「ラ」になってしまいます。

修正方法：舌先をしっかりと歯茎に当てる。

間違い2：舌先が離れる

発音中に舌先が離れてしまうと、Lの音が不明瞭になります。

修正方法：舌先を歯茎に当てたまま、声を出す。

間違い3：口をすぼめる

口をすぼめてしまうと、Rと混同してしまいます。

修正方法：口は自然な形で発音する。

RとLの違いを明確に（詳細版）

RとLの違いを明確にするために、対比して練習しましょう。違いを理解することで、正しい発音ができるようになります。

RとLの比較（詳細）

R：舌根を上げる

  • 舌先はどこにも当てない（下の歯の裏に軽く当てる）
  • 舌の後ろの部分（舌根）を上に上げる
  • 口をすぼめる（「ウ」の形）
  • 唇を前に突き出す
  • 「ルー」という感じ
  • 音が深く響く

L：舌先を歯茎に当てる

  • 舌先を上の歯茎にしっかりと当てる
  • 舌の後ろの部分（舌根）は下げる
  • 口は自然な形
  • 口をすぼめない
  • 「ル」という感じ
  • 音がはっきりしている

違いを感じる練習

練習1：舌の位置を確認

鏡を見ながら、RとLを交互に発音して、舌の位置の違いを確認しましょう。

  • R：舌先が下の歯の裏、舌根が上がっている
  • L：舌先が上の歯茎、舌根が下がっている

練習2：口の形を確認

鏡を見ながら、RとLを交互に発音して、口の形の違いを確認しましょう。

  • R：口がすぼまっている（唇が前に突き出ている）
  • L：口が自然な形

練習3：音の違いを感じる

RとLを交互に発音して、音の違いを感じましょう。

  • R：深く響く音

RとLの位置による違い

RとLは、単語の中での位置によって、発音が少し変わることがあります。これを理解することで、より自然な発音ができます。

Rの位置による違い

語頭のR

語頭のRは、はっきりと発音します。舌根をしっかり上げます。

例：right [raɪt]、rock [rɑk]

語中のR

語中のRも、舌根を上げて発音します。ただし、語頭ほど強くはありません。

例：very [ˈveri]、brother [ˈbrʌðər]

語尾のR

語尾のRも、舌根を上げて発音します。アメリカ英語では特に重要です。

例：car [kɑr]、door [dɔr]

Lの位置による違い

語頭のL（明るいL）

語頭のLは、舌先を歯茎にしっかり当てて発音します。

例：light [laɪt]、love [lʌv]

語中のL（明るいL）

語中のLも、舌先を歯茎に当てて発音します。

例：hello [həˈloʊ]、yellow [ˈjeloʊ]

語尾・子音の前のL（暗いL）

語尾や子音の前のLは、舌先を歯茎に当て、舌根を少し上げます。

例：ball [bɔl]、help [help]

セルフチェック方法

自分のRとLの発音をチェックする方法を学びましょう。

チェック方法

1. 鏡を見ながら発音

鏡を見ながら発音することで、舌の位置と口の形を確認できます。

  • R：舌先が動いていないか？舌根が上がっているか？口がすぼまっているか？
  • L：舌先が歯茎に当たっているか？口が自然な形か？

2. 録音して聞き返す

自分の声を録音して聞き返すことで、客観的に発音をチェックできます。

  • ネイティブの音声と比較する
  • RとLの音の違いを感じる
  • 改善点を見つける

3. 手を当てて確認

手を当てることで、体の動きを確認できます。

  • 舌根に手を当てて、Rで舌根が上がっているか確認
  • 頬に手を当てて、口の動きを確認

4. 対比練習で確認

RとLのペアを交互に発音して、違いを確認します。

  • right/light、rock/lockなどを交互に発音
  • 違いが明確に感じられるか確認

重要ポイントまとめ

1. R：舌根を上げる

舌先はどこにも当てず、舌の後ろの部分を上に上げる。口をすぼめて「ルー」という感じで発音。音が深く響く。

2. L：舌先を歯茎に当てる

舌先を上の歯茎にしっかりと当てる。口は自然な形で「ル」という感じで発音。音がはっきりしている。

3. 明るいLと暗いL

Lには「明るいL」（語頭・語中）と「暗いL」（語尾・子音の前）の2種類があります。位置によって発音が少し変わります。

4. 鏡を見ながら練習

RとLの発音を練習するときは、必ず鏡を見ながら、舌の位置と口の形を確認しましょう。

5. 対比練習が効果的

RとLのペア（right/light, rock/lockなど）を交互に発音することで、違いを明確にできます。

6. 位置による違い

RとLは、単語の中での位置（語頭、語中、語尾）によって、発音が少し変わることがあります。

7. 継続が大切

RとLは難しい音ですが、毎日練習すれば必ずマスターできます。あきらめずに続けましょう。
