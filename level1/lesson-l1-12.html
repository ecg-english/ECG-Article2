12. 音の連結②「リンキング（母音＋母音）」

間に小さな音が挟まる。go away →「ゴゥ(w)アウェイ」

なぜ母音＋母音のリンキングが重要なのか？

英語では、前の単語が母音で終わり、次の単語が母音で始まる場合、間に小さな音（w, y, r）が挟まる現象が起こります。これを理解することで、より自然な英語の発音ができます。

母音＋母音のリンキングが重要な理由

1. より自然な流暢さ

母音＋母音のリンキングをマスターすることで、より自然で流暢な英語の発音ができるようになります。これが「英語らしく聞こえる」鍵です。

2. 聞き取りの鍵

母音＋母音のリンキングを理解することで、ネイティブの話す英語を聞き取りやすくなります。自分で正しく発音できる音は聞き取れるようになります。

3. 滑らかな移行

母音から母音へ直接移行するよりも、間に小さな音を挟むことで、より滑らかに移行できます。

よくある間違い

実際の音：[goʊ əˈweɪ] → 「ゴゥ(w)アウェイ」（間に「w」が挟まる）

リンキング（母音＋母音）とは？（詳細版）

リンキング（母音＋母音）は、前の単語が母音で終わり、次の単語が母音で始まる場合、間に小さな音（w, y, r）が挟まる現象です。英語では、この現象が頻繁に起こり、自然な流暢さを作り出します。

リンキング（母音＋母音）のルール（詳細）

条件：前の単語が母音で終わる + 次の単語が母音で始まる

この条件が揃うと、間に小さな音が挟まります。

挟まる音：w, y, r（前の単語の最後の母音によって決まる）

  • 「o」「u」で終わる → 「w」が挟まる
  • 「i」「e」で終わる → 「y」が挟まる
  • 「r」で終わる（r音化）→ 「r」が挟まる

発音のコツ（詳細）

前の単語の母音から、次の単語の母音へ滑らかに移行します。間に小さな音を挟みます。

重要：自然に、滑らかに発音する

  • 前の単語の母音から、挟まる音を経由して、次の単語の母音へ移行
  • 挟まる音は自然に、滑らかに発音
  • 無理に強調する必要はない

母音＋母音のリンキングが起こる理由

1. 滑らかな移行

母音から母音へ直接移行するよりも、間に小さな音を挟むことで、より滑らかに移行できます。

例：go away → 「ゴゥ(w)アウェイ」と滑らかに移行

2. 自然な流れ

英語では、単語を区切って読むよりも、滑らかにつなげて読む方が自然です。

母音＋母音のリンキングは、英語の自然な流れを作り出します。

3. リズムの維持

母音＋母音のリンキングを使うことで、英語特有のリズム（ストレス・タイム）を維持できます。

単語を区切って読むと、リズムが崩れます。

✓ 挟まる音を確認できているか？

挟まる音の種類（詳細版）

前の単語の最後の母音によって、挟まる音が決まります。このルールを理解することで、正しく発音できるようになります。

1. 「w」が挟まる場合（詳細）

前の単語が「o」や「u」で終わる場合、「w」が挟まります。これは、これらの母音が「w」の音に近いためです。

発音のコツ：

  • 前の単語の「o」や「u」から、自然に「w」の音を出す
  • 「w」は短く、弱く発音
  • 次の単語の母音へ滑らかに移行

例：

  • go away [goʊ əˈweɪ] → 「ゴゥ(w)アウェイ」
  • do it [du ɪt] → 「ドゥ(w)イット」
  • you are [ju ɑr] → 「ユー(w)アー」
  • how about [haʊ əˈbaʊt] → 「ハウ(w)アバウト」
  • show it [ʃoʊ ɪt] → 「ショウ(w)イット」
  • to ask [tu æsk] → 「トゥ(w)アスク」

2. 「y」が挟まる場合（詳細）

前の単語が「i」や「e」で終わる場合、「y」が挟まります。これは、これらの母音が「y」の音に近いためです。

発音のコツ：

  • 前の単語の「i」や「e」から、自然に「y」の音を出す
  • 「y」は短く、弱く発音
  • 次の単語の母音へ滑らかに移行

例：

  • see it [si ɪt] → 「シー(y)イット」
  • say it [seɪ ɪt] → 「セイ(y)イット」
  • the end [ði end] → 「ジー(y)エンド」
  • my idea [maɪ aɪˈdiə] → 「マイ(y)アイディア」
  • we are [wi ɑr] → 「ウィー(y)アー」
  • they are [ðeɪ ɑr] → 「ゼイ(y)アー」

3. 「r」が挟まる場合（詳細）

前の単語が「r」で終わる場合（r音化）、「r」が挟まります。これは、r音化英語（アメリカ英語など）の特徴です。

発音のコツ：

  • 前の単語の「r」から、自然に「r」の音を出す
  • 「r」は短く、弱く発音
  • 次の単語の母音へ滑らかに移行

例：

  • far away [fɑr əˈweɪ] → 「ファー(r)アウェイ」
  • more of [mɔr əv] → 「モー(r)アブ」
  • here is [hɪr ɪz] → 「ヒア(r)イズ」
  • for a [fɔr ə] → 「フォー(r)ア」
  • there is [ðer ɪz] → 「ゼア(r)イズ」
  • where is [wer ɪz] → 「ウェア(r)イズ」

挟まる音の見分け方（まとめ表）

重要ポイントまとめ

1. 母音＋母音のリンキング

前の単語の母音から、挟まる音を経由して、次の単語の母音へ滑らかに移行します。これが母音＋母音のリンキングの基本です。

4. 自然に発音

挟まる音は、自然に、滑らかに発音します。無理に強調する必要はありません。挟まる音を強調しすぎると、不自然に聞こえます。

5. フレーズで練習

母音＋母音のリンキングをマスターするには、フレーズで練習することが重要です。単語だけでなく、フレーズで練習しましょう。例：go away, see it, far away

6. 対比練習が効果的

区切って読む場合とつなげて読む場合を対比して練習することで、違いを明確にできます。例：「ゴー アウェイ」（区切って）→ 「ゴゥ(w)アウェイ」（つなげて）

母音から母音へ直接移行するよりも、間に小さな音を挟むことで、より滑らかに移行できます。これが母音＋母音のリンキングが起こる理由です。

8. 継続が大切

母音＋母音のリンキングは難しいですが、毎日練習すれば必ずマスターできます。あきらめずに続けましょう。
