8. 単語の「ストレス（アクセント）」の位置

強く読むところ、弱く読むところ。photograph / photographer

なぜストレスが重要なのか？

英語では、単語の中に「強く読む音節」と「弱く読む音節」があります。この「強弱」が、英語らしいリズムを作り出します。ストレスを間違えると、ネイティブには通じにくくなります。

ストレスが重要な理由

1. 英語らしいリズムを作る

ストレスを正しく理解することで、英語特有の「ストレス・タイム（強勢拍子）」のリズムが生まれます。これが「英語らしく聞こえる」最大の鍵です。

2. 意味を区別する

ストレスの位置が違うと、意味が変わる場合があります。例：record（名詞）とrecord（動詞）では、ストレスの位置が違います。

3. 聞き取りの鍵

ストレスを理解することで、ネイティブの話す英語を聞き取りやすくなります。自分で正しく発音できる音は聞き取れるようになります。

よくある間違い

間違い1：すべての音節を均等に発音する

photograph（写真）

カタカナ読み：「フォトグラフ」（均等なリズム）

実際の音：[ˈfoʊtəgræf] → 最初の音節「フォ」を強く読む

修正方法：最初の音節を強く、他を弱く発音する

間違い2：ストレスの位置を間違える

photographer（写真家）

カタカナ読み：「フォトグラファー」（均等なリズム）

実際の音：[fəˈtɑgrəfər] → 2番目の音節「ト」を強く読む

同じ語源なのに、ストレスの位置が違う！

修正方法：2番目の音節を強く、他を弱く発音する

間違い3：名詞と動詞でストレスを区別しない

record（名詞）とrecord（動詞）で、ストレスの位置を区別しない。

修正方法：名詞は最初の音節、動詞は2番目の音節を強く発音する

日本語と英語の違い

🇯🇵 日本語

  • すべての音節を均等に発音
  • 強弱のリズムが少ない
  • ストレスが存在しない
  • 「バナナ」→ 3音節を均等に

🇺🇸 英語

  • 強弱のリズムが重要
  • ストレスがある音節を強く
  • ストレスがない音節を弱く
  • 「banana」→ 真ん中を強く、他を弱く

ストレスを正しく理解することで、英語らしいリズムと強弱が生まれます。これが「英語らしく聞こえる」鍵です。このレッスンで、ストレスを完全にマスターしましょう。

ストレスとは？（詳細版）

ストレス（アクセント）は、単語の中で「強く読む音節」のことです。英語では、1つの単語に必ず1つのストレスがあります。ストレスを理解することが、英語らしい発音への第一歩です。

ストレスの特徴（詳細）

1. 強く読む音節

ストレスがある音節は、他の音節より「強く、長く、高く」読まれます。

例：banana [bəˈnænə] → 「ナ」を強く読む

  • 強く：他の音節より大きな声で
  • 長く：他の音節より長く伸ばす
  • 高く：他の音節より高い音程で

2. 弱く読む音節

ストレスがない音節は、「弱く、短く、低く」読まれます。多くの場合、シュワ音 [ə] になります。

例：banana [bəˈnænə] → 「バ」と最後の「ナ」を弱く読む

  • 弱く：ストレスがある音節より小さな声で
  • 短く：ストレスがある音節より短く
  • 低く：ストレスがある音節より低い音程で

3. 1つの単語に1つのストレス

英語では、1つの単語に必ず1つのストレスがあります（長い単語では2つ以上の場合もあります）。

主ストレス（第一強勢）と副ストレス（第二強勢）がある場合もあります。

ストレスの3つの要素

1. 強さ（Loudness）

ストレスがある音節は、他の音節より大きな声で読まれます。

例：banana → 「ナ」を大きく読む

2. 長さ（Length）

ストレスがある音節は、他の音節より長く読まれます。

例：banana → 「ナ」を長く読む

3. 高さ（Pitch）

ストレスがある音節は、他の音節より高い音程で読まれます。

例：banana → 「ナ」を高く読む

ストレスの位置のルール（詳細版）

ストレスの位置には、いくつかのルールがあります。ただし、例外も多いので、単語ごとに覚えることが重要です。ルールを理解することで、多くの単語のストレスを予測できるようになります。

1. 2音節の名詞・形容詞

多くの場合、最初の音節にストレスがあります。これは英語の基本的なルールです。

例：

  • apple [ˈæpəl] → 「ア」を強く
  • happy [ˈhæpi] → 「ハ」を強く
  • table [ˈteɪbəl] → 「テ」を強く
  • water [ˈwɔtər] → 「ウォ」を強く
  • paper [ˈpeɪpər] → 「ペ」を強く
  • mother [ˈmʌðər] → 「マ」を強く

⚠ 例外

  • hotel [hoʊˈtel] → 2番目の音節にストレス
  • police [pəˈlis] → 2番目の音節にストレス

2. 2音節の動詞

多くの場合、2番目の音節にストレスがあります。名詞・形容詞とは逆のパターンです。

例：

  • begin [bɪˈgɪn] → 「ギ」を強く
  • forget [fərˈget] → 「ゲ」を強く
  • decide [dɪˈsaɪd] → 「サイ」を強く
  • arrive [əˈraɪv] → 「ライ」を強く
  • believe [bɪˈliv] → 「リ」を強く
  • complete [kəmˈplit] → 「プ」を強く

⚠ 例外

  • answer [ˈænsər] → 最初の音節にストレス
  • enter [ˈentər] → 最初の音節にストレス

3. 3音節以上の単語（語尾によるルール）

語尾によってストレスの位置が決まることが多いです。語尾のパターンを覚えると便利です。

-tion で終わる：

  • その前の音節にストレス
  • nation [ˈneɪʃən]
  • action [ˈækʃən]
  • education [ˌedʒuˈkeɪʃən]

-ic で終わる：

  • その前の音節にストレス
  • music [ˈmjuzɪk]
  • basic [ˈbeɪsɪk]
  • economic [ˌikəˈnɑmɪk]

-al で終わる：

  • その前の音節にストレス
  • animal [ˈænəməl]
  • hospital [ˈhɑspɪtəl]
  • general [ˈdʒenərəl]

-er, -or, -ar で終わる：

  • その前の音節にストレス
  • teacher [ˈtitʃər]
  • doctor [ˈdɑktər]
  • dollar [ˈdɑlər]

4. 接頭辞・接尾辞の影響

接頭辞（un-, re-, pre- など）や接尾辞（-ly, -ful, -less など）は、ストレスの位置に影響を与えることがあります。

接頭辞の例：

  • unhappy [ʌnˈhæpi] → 接頭辞「un」は弱く、元の単語のストレスを維持
  • return [rɪˈtɜrn] → 接頭辞「re」は弱く、2番目の音節にストレス

接尾辞の例：

  • quickly [ˈkwɪkli] → 接尾辞「-ly」は弱く、元の単語のストレスを維持
  • careful [ˈkerfəl] → 接尾辞「-ful」は弱く、元の単語のストレスを維持

ストレスの位置が変わる例（詳細版）

同じ語源でも、品詞が変わるとストレスの位置が変わる場合があります。これを理解することで、正しく発音できるようになります。

ストレスの位置が変わる例（詳細）

photograph [ˈfoʊtəgræf]（名詞：写真）

ストレス：最初の音節「フォ」

強：フォ、弱：トグラフ

photographer [fəˈtɑgrəfər]（名詞：写真家）

ストレス：2番目の音節「ト」

同じ語源なのに、ストレスの位置が違う！

強：ト、弱：フォグラファー

ストレス：2番目の音節「ト」

強：ト、弱：フォグラフィ

record [ˈrekərd]（名詞：記録）

ストレス：最初の音節「レ」

強：レ、弱：コード

record [rɪˈkɔrd]（動詞：記録する）

ストレス：2番目の音節「コ」

同じ単語でも、品詞が違うとストレスの位置が変わる！

強：コ、弱：リ

present [ˈprezənt]（名詞：贈り物）

ストレス：最初の音節「プレ」

強：プレ、弱：ゼント

present [prɪˈzent]（動詞：提示する）

ストレス：2番目の音節「ゼ」

同じ単語でも、品詞が違うとストレスの位置が変わる！

強：ゼ、弱：プリ

ストレスの位置が変わる理由

1. 品詞による違い

名詞は最初の音節、動詞は2番目の音節にストレスがあることが多いです。

例：record（名詞）は最初、record（動詞）は2番目

2. 語尾による違い

語尾が変わると、ストレスの位置も変わることがあります。

例：photograph → photographer（語尾が変わる）

ストレスの見つけ方（詳細版）

辞書でストレスの位置を確認する方法を学びましょう。辞書を活用することで、正しいストレスを覚えることができます。

辞書での表記（詳細）

1. 発音記号の前の記号

発音記号の前に ˈ（ストレス記号）がついている音節がストレスです。

例：banana [bəˈnænə] → 「ナ」の前にストレス記号がないので、ストレスは「ナ」

実際は [bəˈnænə] と表記され、「ナ」がストレス

主ストレス（第一強勢）は ˈ、副ストレス（第二強勢）は ˌ で表記されます。

2. カタカナ表記の太字

カタカナ表記で太字になっている部分がストレスです。

例：banana → 「バナナ」→ 「ナ」がストレス

多くの日本語の辞書では、太字でストレスを表記しています。

3. オンライン辞書の活用

オンライン辞書（Merriam-Webster、Oxford、Cambridge など）では、音声と発音記号を確認できます。

  • 音声を聞いて、ストレスの位置を確認
  • 発音記号で、ストレスの位置を確認
  • 何度も聞いて、ストレスを覚える

ストレスを見つけるコツ

1. 音声を聞く

ネイティブの音声を聞いて、どの音節が強く読まれているかを確認します。

強く読まれている音節がストレスです。

2. 手を叩いてリズムを感じる

強く読む音節で手を強く叩き、弱く読む音節で手を軽く叩いて、リズムを感じます。

強く叩く音節がストレスです。

3. ルールを活用する

2音節の名詞・形容詞は最初、2音節の動詞は2番目など、ルールを活用します。

ただし、例外も多いので、辞書で確認することが重要です。

練習4：手を叩いてリズムを感じる

ステップ1：強く読む音節で手を強く叩く

ストレスがある音節で手を強く叩きます。

例：banana → 真ん中の「ナ」で強く叩く

ステップ2：弱く読む音節で手を軽く叩く

例：banana → 最初の「バ」と最後の「ナ」で軽く叩く

ステップ3：リズムを感じながら発音

手を叩きながら発音して、強弱のリズムを感じます。

強弱のリズムを感じることで、ストレスを正しく発音できるようになります。

練習5：ストレスを含む文で練習

短い文で練習

  • I want an apple.（私はリンゴが欲しい）
  • I see a teacher.（私は教師を見ます）
  • I begin today.（私は今日始めます）

文の中でストレスを正しく発音する練習をしましょう。

長い文で練習

  • I want to record a record.（私は記録を記録したいです）
  • I see a photographer taking a photograph.（私は写真家が写真を撮っているのを見ます）

長い文でも、ストレスを正しく発音する練習をしましょう。

よくある間違いとその修正方法

ストレスの発音でよくある間違いと、その修正方法を学びましょう。

症状

セルフチェック方法

自分のストレスの発音をチェックする方法を学びましょう。

チェック方法

1. 強弱のリズムを確認

強く読む音節と弱く読む音節のリズムを確認します。

  • 強く読む音節は強く発音できているか？
  • 弱く読む音節は弱く発音できているか？
  • 強弱の違いが明確か？

2. 録音して聞き返す

自分の声を録音して聞き返すことで、客観的に発音をチェックできます。

  • ネイティブの音声と比較する
  • ストレスの位置が正しいか確認
  • 強弱のリズムが感じられるか確認
  • 改善点を見つける

3. 手を叩いてリズムを確認

強く読む音節で手を強く叩き、弱く読む音節で手を軽く叩いて、リズムを確認します。

  • banana：軽く・強く・軽く
  • today：軽く・強く
  • リズムが感じられるか確認

4. 辞書で確認

辞書でストレスの位置を確認し、自分の発音と比較します。

  • 発音記号でストレスの位置を確認
  • 音声を聞いて、ストレスの位置を確認
  • 自分の発音と比較する

聞き取りの練習

ストレスを正しく発音するだけでなく、聞き取ることも重要です。聞き取りの練習もしましょう。

聞き取りの練習方法

1. ネイティブの音声を聞く

ネイティブの音声を聞いて、ストレスと強弱のリズムを感じましょう。

  • banana, today, photographなどの単語を聞く
  • ストレスが強く発音されていることを感じる
  • 強弱のリズムを感じる
  • 何度も繰り返し聞く

2. 聞き取りクイズ

ストレスを含む単語を聞いて、ストレスの位置を判断する練習をしましょう。

  • どの音節が強く読まれているか？
  • どの音節が弱く読まれているか？
  • 強弱のリズムを感じられるか？

3. シャドーイング

ネイティブの音声を聞きながら、すぐ後に続いて発音する練習をしましょう。

  • ストレスを含む文を聞く
  • すぐ後に続いて発音する
  • ストレスを正しく発音することを意識する
  • 強弱のリズムを意識する

継続的な練習のコツ

ストレスをマスターするには継続が最も重要です。続けやすいコツを学びましょう。

継続のコツ

1. 毎日少しずつ練習する

1日5分でも10分でも、毎日続けることが大切です。完璧を目指さず、継続を優先しましょう。

2. 強弱のリズムを意識する

ストレスをマスターするには、強弱のリズムを意識することが重要です。手を叩いてリズムを感じましょう。

3. 対比練習を繰り返す

強く読む音節と弱く読む音節を対比して練習することで、違いを明確にできます。

4. 楽しみながら練習する

好きな歌を歌ったり、好きな映画のセリフを真似したり、楽しみながら練習しましょう。

5. 小さな目標を設定する

「1週間続ける」「1つの単語をマスターする」など、小さな目標を設定して達成感を味わいましょう。

重要ポイントまとめ

1. ストレスは最重要

ストレスを正しく理解することで、英語らしいリズムと強弱が生まれます。これが「英語らしく聞こえる」鍵です。英語の母音の約半分がシュワ音で、ストレスと組み合わせることで英語らしいリズムが生まれます。

2. 強く読む音節と弱く読む音節

ストレスがある音節は「強く、長く、高く」、ストレスがない音節は「弱く、短く、低く」読まれます。ストレスの3つの要素（強さ、長さ、高さ）を意識することが重要です。

3. ストレスの位置のルール

2音節の名詞・形容詞は最初の音節、2音節の動詞は2番目の音節にストレスがあることが多いです。3音節以上の単語は、語尾（-tion, -ic, -al, -er, -or, -ar など）によってストレスの位置が決まることが多いです。ただし、例外も多いので、単語ごとに覚えることが重要です。

4. ストレスの位置が変わる例

同じ語源でも、品詞が変わるとストレスの位置が変わる場合があります（例：photograph / photographer / photography、record（名詞）/ record（動詞）、present（名詞）/ present（動詞））。

5. 辞書で確認

発音記号の前の ˈ（ストレス記号）や、カタカナ表記の太字でストレスの位置を確認できます。オンライン辞書では、音声と発音記号を確認できます。

6. 手を叩いてリズムを感じる

強く読む音節で手を強く叩き、弱く読む音節で手を軽く叩いて、リズムを感じる練習が効果的です。強弱のリズムを感じることで、ストレスを正しく発音できるようになります。

7. 対比練習が効果的

強く読む音節と弱く読む音節を対比して練習することで、違いを明確にできます。また、ストレスの位置が変わる例（photograph / photographer）を対比して練習することも効果的です。

8. 継続が大切

ストレスは難しいですが、毎日練習すれば必ずマスターできます。あきらめずに続けましょう。
